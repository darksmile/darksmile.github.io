How to blog from android device

Doctor's code
2017-01-30

This article will help you to start blog from your phone.
Used technologies: Android device, termux, hexo, vim (or Emacs) and github pages.

Motivation

Just imagine that you’re about 1000 years old, and your race is one of the oldest and smartest races in galaxy. You saw the Roman Empire raising and the End of the Universe. Sure, you want to share your knowledge. But what if writing posts is boring and distracts you from drinking tea with the Queen Elizabeth and fighting Cybermen in parallel universe?
You can use your psychic paper phone to blog whenever you have a free second!

Termux - linux in your phone

To start, you need to install termux - terminal emulator and linux environment for android.

When installed, you can see a list of available packages and install you favorite.

Node.js and hexo

I wanted to use a static blog engine to serve it using github pages, and I had working nodejs in termux. That’s why I chose hexo.

First, install node.js

And hexo client

And hexo server to serve locally and see the result

Create your blog

Now open http://localhost:4000 and voilà, this is your blog!

Writing posts

Now, you need to edit text files on your phone. There are two problems here:

  • sdcard has fat32 (or other fat) file system, which is not supporting symlinks. But they are needed to install npm packages;
  • external apps can’t use other’s internal storage.

Think, Doctor, think!.. I know! Sonic screwdriver Emacs! We can use Emacs (or vim) inside termux.

Install Emacs

Now you can edit config and post files with Emacs.

Note: you need an extended keyboard to use Emacs or vim on android. Hacker’s keyboard will help you.

Publish to github pages

Next step - publish. We have to setup git and github.

Generate an ssh key with termux

Go to your github account (settings -> keys) and add this key.

Install and configure git

Create github repository for github pages. It should be named like <your github nickname>.github.io. More details here.

Time to configure your hexo to deploy to github.

Firstly, install hexo-deployer-git

Okay, we’re ready. Let’s do it!

Open https://<your github nickname>.github.io and enjoy!

May the force be with you, young padawan! Oops, this is from another story.

See you later!

Allons-y!

Yours, Doctor
